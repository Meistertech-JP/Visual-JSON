# Visual JSON Samples

These synthetic files are designed to show Visual JSON's core workflows quickly.

- `api-response-invalid.json`: open it, load `schema/user.schema.json`, run schema validation, then click the diagnostics to jump to the failing values.
- `users-array.json`: switch to Grid, open the object array as Table View, sort columns, and edit a scalar cell.
- `config.jsonc`: try JSONC comments, trailing commas, validation, and formatting.
- `events.jsonl`: open JSON Lines as an array-equivalent document and save it back as line format.
