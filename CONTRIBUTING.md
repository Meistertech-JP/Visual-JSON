# Contributing

External contributions to Visual JSON are accepted under the Developer Certificate of Origin (DCO), not a separate contributor license agreement.

By contributing, you certify that you have the right to submit the contribution and that it may be distributed under this project's license.

Every commit must include a sign-off line:

```text
Signed-off-by: Your Name <your.email@example.com>
```

The usual way to add it is:

```powershell
git commit -s
```

The sign-off means you certify the Developer Certificate of Origin 1.1: https://developercertificate.org/
